Drupal note:

Aligned fields

The field aligned modifier should be used for aligning fields that do not have a label with fields that do have a label.

.field {
  margin-bottom: 0.556em;

  + .field--aligned,
  &:last-child {
    margin-bottom: 0;
  }

  .field__item--expired {
    text-decoration: line-through;
  }

  .label-above {
    @extend .ecl-h3;
  }

  .field__items {
    p:last-child,
    ul:last-child {
      margin-bottom: 0 !important;
    }
  }

  .field__label {
    font-size: map-get($ecl-font-size, 'xs');
    font-weight: 600;

    h4 {
      padding: 0;
    }

    a:focus {
      outline-offset: 2px;
    }
  }

  // @todo: Why do we have this html? (body field in priorities)
  .sections .container {
    padding: 0;
  }
}
